Niveau 2, 300m²

body: >-


Ici, rien n’est figé. Le décor change selon les projets. Le foyer moderne accueille habituellement le bar pour les concerts. Les couleurs des projecteurs font monter la chaleur, les miroirs reflètent les idées, l’agencement se plie aux appétences du moment. Ambiance tamisée, fête endiablée ou cocktail élégant, dans le Foyer moderne, tout est magiquement possible.
